# Rule of 100 — Daily Accountability Dashboard

A dark-mode daily accountability tracker built around Alex Hormozi's Rule of 100 — log 100 outreach actions every single day across any mix of methods.

**Live:** https://rule-of-100.netlify.app

---

## What's Inside

### Action Tracking (Rule of 100)
Single 100-action pool across Cold Outreach, Warm Outreach, Content Creation, and Paid Ads. Per-platform logging, real-time progress ring, grade (F→S), XP system, and confetti at milestones.

### Outreach Trackers
- **Doors Knocked** — knocked / answered / appointments with live conversion rates and daily goal
- **Email Campaigns** — sent / opened / replied with open rate, reply rate, and daily goal

### Lead Pipeline (Kanban)
5 stages: Contact → Engaged → Demo → Won → Closed. Drag and drop cards, value tracking per prospect, pipeline summary with total revenue won and active deal count.

### Weekly Summary
7-day bar chart, total actions, days hit, daily average, top method, plain-English insight.

### 28-Day History Grid
Color-coded calendar (S/A/B/C/D/F grades), streak tracking.

### Gamification
XP levels (Rookie → Grinder → Hustler → Closer → Machine → Legend), streak badges at 3/7/14/21/30/60/100 days, mega confetti on milestones.

### Daily Debrief
What Worked / Do Better Tomorrow / Tomorrow's Focus — auto-saved, past 14 entries viewer.

### Light / Dark Mode
One-tap toggle, saved across sessions.

---

## Tech
Pure HTML/CSS/JS — no framework, no build step, no backend. All data in localStorage. Open `index.html` in any browser to run locally.

Hosted on Netlify — push to main = auto-deploy.
